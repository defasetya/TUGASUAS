Deskripsi singkat

Web Asiquefest adalah sebuah web penyedia informasi event musik dan bertujuan untuk memudahkan
penggunanya dalam mencari informasi mengenai event musik yang akan datang.

*******************
Tools

Untuk Frontend gunakan Javascript, HTML, CSS dan menggunakan Framework Code Igniter 3
Untuk Backend gunakan Javascript, HTML, CSS, dan menggunakan Framework CodeIgniter 3

*******************

Spesifikasi Web

Backend :
 Login page
 Register page
 Dashboard page

Frontend :
 Landing Page/Home Page
 About page
 Concert Page
 Gallery page
 Contact page

Kata Kunci

HTTP Methods : get,post
CSS : margin, padding, font-size, text-align, flex, grid, border, color, div, span
